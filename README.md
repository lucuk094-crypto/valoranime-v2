# 🎬 VALORANIME

Website streaming anime & novel dengan sistem admin management menggunakan Supabase.

## 📚 DOKUMENTASI

**Baca panduan lengkap di:** [`PANDUAN-LENGKAP.md`](./PANDUAN-LENGKAP.md)

Panduan tersebut berisi:
- ✅ Setup dari awal
- ✅ Cara tambah admin/user
- ✅ Cara ganti password
- ✅ Cara update level & EXP user
- ✅ Cara update role, ban/unban user
- ✅ Deploy ke Vercel
- ✅ Troubleshooting lengkap

## ⚡ QUICK START

```bash
# Install dependencies
npm install

# Setup environment variables
# Edit .env.local dengan Supabase keys Anda

# Start development server
npm run dev

# Buat admin pertama
npm run quick-admin

# Login di http://localhost:3000/admin/login
```

## 🎯 COMMANDS

```bash
npm run dev              # Start development server
npm run quick-admin      # Buat admin cepat
npm run create-admin     # Buat admin interaktif
npm run reset-password   # Reset password user
npm run test-supabase    # Test koneksi Supabase
```

## 📦 TECH STACK

- **Framework**: Next.js 15
- **Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## 🔗 LINKS

- **Admin Login**: http://localhost:3000/admin/login
- **Supabase Dashboard**: https://supabase.com/dashboard
